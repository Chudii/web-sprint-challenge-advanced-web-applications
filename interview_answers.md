1. Explain what a token is used for.
A token is a reserved text component in javascript that is used  for the specification of logic within an application. For instance, "var" and "function" are tokens that each serve their designated purpose.

2. What steps can you take in your web apps to keep your data secure?
You can upload your data to a database.


3. Describe how web servers work.
Web servers process and respond to requests given by local clients and upon  receiving the request, the server checks to see if there is an existing file. If there is none, an application builds it.
